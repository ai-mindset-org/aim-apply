философия

мы не ищем сотрудников. мы ищем людей, которые уже живут внутри AI-first парадигмы.

/apply -- это не анкета. это рабочая среда. ты клонируешь репо, запускаешь агента, решаешь задачу. если тебе это привычно -- ты уже внутри.

нет резюме. нет сопроводительных писем. только git log и то, как ты думаешь рядом с машиной.

последняя вакансия -- потому что после неё ты либо работаешь с нами, либо понимаешь, что можешь строить что угодно сам.

как это работает
aim-apply
$ git clone https://github.com/ai-mindset-org/aim-apply
$ cd aim-apply
$ claude
# агент читает AGENTS.md, даёт тебе задачу
# ты решаешь, коммитишь, пушишь
 
$ git push origin my-solution
$ gh pr create
# готово. PR = твоя заявка

никаких форм. pull request -- это и есть отклик. код говорит за тебя.

scoring

каждый PR оценивается по 4 осям, каждая от 0 до 4:

thinking
craft
velocity
signal
0-4 как ты разбираешь задачу декомпозиция, edge cases, решения
0-4 качество результата код, структура, git hygiene
0-4 скорость и итерации agent usage, automation, workflow
0-4 что ещё видно инициатива, стиль, потенциал

12+ из 16 -- приглашение на разговор. 14+ -- оффер.
