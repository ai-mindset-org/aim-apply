о роли

не контент-менеджер. не SMM. не «ведёт соцсети».

нам нужен AI-native маркетолог-дженералист, который:

  • считает результат маркетинга в деньгах, а не в лайках
  • формулирует эксперименты каждую неделю
  • открывает новые каналы до того, как текущие насытятся
  • жонглирует контентом, аналитикой и закупками одновременно

со временем это вырастет в команду. но сейчас нужен один человек, который выстроит фундамент.

чем занимаешься
01 performance & эксперименты 50%

это главное. всё остальное поддерживает это.

  • дизайн, запуск и анализ маркетинговых экспериментов каждую неделю
  • построение и оптимизация воронок от первого касания до оплаты
  • настройка и ведение рекламы — TG Ads, Meta, Google (хотя бы 2 из 3)
  • управление рекламным бюджетом с ROI-целями по каждому каналу
  • постоянный поиск новых каналов — текущие насытятся
  • weekly report: расходы, лиды, конверсии, выручка
02 контент для конверсии 30%

не творчество ради творчества. контент — инструмент привлечения.

  • переупаковка экспертного контента (видео, подкасты, доклады) в рекламные форматы
  • конверсионный копирайтинг для TG, ботов, лендингов, email
  • рекламные креативы — тексты и визуал
  • A/B тестирование посадочных страниц
  • нарезка видео под короткие форматы (reels, shorts)
03 market intelligence 20%
  • мониторинг рынка: конкуренты, тренды в edtech/AI, инфоповоды
  • channel discovery: исследование новых площадок раньше конкурентов
  • еженедельный дайджест для команды
  • автоматизация рутины через AI-инструменты
обязательные требования

в порядке приоритета. первые три блока — deal-breakers.

priority 01 · performance marketing
фундамент — без него AI-скиллы строить не на чем
  • 5+ лет опыта в performance/digital marketing — edtech, e-com или B2C SaaS
  • дизайн экспериментов — гипотеза → метрика → тест → анализ → следующий тест
  • аналитика — воронки, конверсии, когортный анализ, unit economics, LTV, CAC
  • управление бюджетом — опыт от $5K/мес, ROI-цели по каналу
  • закупка трафика — TG Ads, Meta Ads, Google Ads — самостоятельное ведение
deal-breaker
priority 02 · AI-native (главное в названии роли)
AI — не "инструмент на потом", а твой ежедневный рабочий стек
  • ежедневное использование AI для реальной работы, не для фана — Claude, ChatGPT, Cursor, Codex, собственные workflows. если ты открываешь ChatGPT "иногда для текстов" — это не то.
  • собственные workflows — ты строишь свои пайплайны, не используешь стандартные интерфейсы. например: скрипт парсит выгрузку Meta Ads → считает когорты → генерирует инсайты → кидает в Telegram
  • AI в аналитике кампаний — не "AI пишет отчёт", а AI помогает найти паттерн в данных, который ты не заметил
  • prompt engineering как навык — формулировка задачи так, чтобы артефакт был рабочим с первого раза
  • code-AI — Claude Code / Cursor / Copilot для аналитики, автоматизации, ботов. не обязательно писать production-код, но должен уметь скорректировать скрипт под свою задачу
  • опыт создания AI-powered воронок, ботов или workflows — сильный плюс
deal-breaker
priority 03 · мышление
как ты подходишь к задачам
  • channel discovery — постоянно ищешь новые каналы, не только оптимизируешь старые
  • инициативность — предлагаешь эксперименты сам, не ждёшь задач
  • быстрая итерация — запустил → измерил → скорректировал
  • data-driven — каждое решение подкреплено данными
deal-breaker
priority 04 · контент и копирайтинг
язык, которым ты пишешь
  • русский — основной язык, лаконичный стиль, без корпоративщины
  • английский — рабочий уровень
  • конверсионный копирайтинг — тексты, которые продают
стек команды

инструменты, с которыми ты будешь работать

мы не ищем "знание конкретных инструментов". мы ищем способ мышления. но если тебе интересно, на чём мы строим операционку — вот наш стек. всё, что можно, написано нами же.

Claude Code основной

главный инструмент всей команды. 100+ собственных skills, MCP-интеграции с Linear, Calendar, Telegram, Notion, Sheets. у каждого — свои workflows для аналитики, контента, автоматизации.

Obsidian knowledge

vault на ~12K файлов — research, meeting notes, транскрипты, контент, стратегия. общий контекст шарится между агентами через Dropbox Sync.

Cursor / Codex AI IDE

для кода, лендингов, длинных документов. multi-file editing, inline-AI, tab completion там, где Claude Code избыточен. некоторые используют Copilot/Codex как second-opinion на сложный скрипт.

ChatGPT быстрые задачи

отдельный инструмент, не "запасной Claude". используем, когда задача точно сформулирована и нужен быстрый артефакт — копирайт, формулировка, brainstorm.

Linear tasks

все проекты, циклы, milestones. single source of truth для работы команды — интегрирован с Claude Code через MCP.

Telegram comms

внутренняя коммуникация, каналы, боты. собственные MCP-скрипты для автоматизации постов, рассылок, аналитики аудитории.

N8N automation

визуальные пайплайны между сервисами. когда нужен scheduled workflow или webhook trigger, который работает без участия человека.

Meta / TG Ads / Google performance

активные рекламные кабинеты. аналитика когорт и воронок — через собственные Claude Code скиллы, не dashboards из коробки.

ты не обязан(а) знать весь стек на старте. но должен(а) быть готов(а) освоить его — потому что команда работает через эти инструменты, а не параллельно им.
